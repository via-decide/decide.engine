Your decision assistant, planning companion, and ViaDecide site guide — powered by Google Gemini. Click the button in the bottom-right to open the widget.

Features

Smart Chat

Ask questions about decisions, policy, research, or any ViaDecide tool.

✅

Track action items and export them alongside your chat to a .docx file.

Quick Start

1

Get a Gemini API Key

Visit Google AI Studio and create a free API key. No credit card needed for the free tier.

2

Open the Agent Widget

Click the purple button in the bottom-right corner of this page to open the agent panel.

3

Enter Your Key

Go to the Guide tab inside the widget, paste your key in the field, and click Save. Then hit Test to verify the connection.

4

Start Chatting

Switch to the Chat tab and ask anything. Your key is stored in localStorage — never sent to our servers.

Privacy & Security

Your Gemini API key is stored in your browser's localStorage and sent directly to Google's API — it never touches ViaDecide servers. Your chat history exists only in memory and is cleared when you refresh. You can clear your key at any time from the Guide tab.

Embedding on Any Page

One-line install

Add this before </body> to embed the widget on any ViaDecide page:

<script src="/js/viadecide-agent.js"></script>

The script is idempotent — including it twice has no effect.
