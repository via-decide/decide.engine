Install our store as a Progressive Web App — no App Store needed. Works on Android, iPhone, Windows, Mac, and Linux. Browse and order offline, loads instantly from your homescreen.

📱 Android — Chrome
  1. Open store.viadecide.com in Chrome
  2. Tap the 3-dot menu (⋮) top-right
  3. Tap "Install app"
  4. Tap Install — icon appears on homescreen
🍎 iPhone / iPad — Safari
  1. Open in Safari (not Chrome)
  2. Tap the Share button (□↑) at bottom
  3. Tap "Add to Home Screen"
  4. Tap Add — done!
💻 Desktop — Chrome / Edge
  1. Open in Chrome or Edge
  2. Look for the install icon ⊕ in address bar
  3. Or: 3-dot menu → "Install PrintByDD"
  4. Click Install — opens as its own app!
